ML Algorithm Showcase

Barbie vs Oppenheimer: IMDB Review Classifier

Skills Utilized

Python Pandas NumPy Scikit-learn TensorFlow Keras BERT Natural Language Processing Data Cleaning Feature Engineering Machine Learning Deep Learning Ensemble Methods Data Visualization Matplotlib Seaborn

Description

This project aims to classify movie reviews as either about "Barbie" or "Oppenheimer" using review data from IMDb provided on Kaggle. The dataset includes textual reviews and 1-10 scale scores. The objective is to identify which movie a given review is for using supervised learning models like linear regression or deep learning. Additionally, we plan to extract new features like word count and average word length and might use unsupervised learning to find useful patterns. A potential challenge is the mention of both movies in reviews, necessitating more thorough data cleaning.

The Algorithm

Binary Classification for IMDB Reviews

This machine learning algorithm solves a binary classification problem, determining whether an IMDB review is for the movie "Barbie" or "Oppenheimer". The project achieves over 90% accuracy using ensemble and decision tree algorithms.

Key Features:

  • Tokenization to use words as features
  • Additional variables like review length
  • Ensemble model combining BERT, Bag of Words, and ANN
  • Decision tree classifier for interpretability

Data Processing:

  • Cleaning and preprocessing of IMDB review data
  • Feature extraction from review text and metadata
  • Data exploration and visualization

Model Architecture:

  • BERT (Bidirectional Encoder Representations from Transformers)
  • Bag of Words with Decision Tree
  • Artificial Neural Network (ANN)
  • Custom ensemble model combining the above
