The Algorithm

Binary Classification for IMDB Reviews

This machine learning algorithm solves a binary classification problem, determining whether an IMDB review is for the movie "Barbie" or "Oppenheimer". The project achieves over 90% accuracy using ensemble and decision tree algorithms.

Key Features:

  • Tokenization to use words as features
  • Additional variables like review length
  • Ensemble model combining BERT, Bag of Words, and ANN
  • Decision tree classifier for interpretability

Data Processing:

  • Cleaning and preprocessing of IMDB review data
  • Feature extraction from review text and metadata
  • Data exploration and visualization

Model Architecture:

  • BERT (Bidirectional Encoder Representations from Transformers)
  • Bag of Words with Decision Tree
  • Artificial Neural Network (ANN)
  • Custom ensemble model combining the above
# Ensemble model architecture
